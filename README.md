# Vanishing Points — Atlas of the Forgotten

An interactive web application mapping haunted and abandoned places around the world. Built with Node.js, Express, SQLite, and Leaflet.js.

## Features

- **100+ curated places** across 11 categories
- **Interactive dark-themed map** with intensity-based markers
- **User submissions** with admin approval workflow
- **Explored marking** and **favorites** (stored locally)
- **Shareable place links**
- **Ambient sound** toggle
- **Responsive design** for mobile and desktop
